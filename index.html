Tugas 3:fronted - HTML

SILVIA WINALDA: Ringkasan Materi 100 Kata

HTML (Hypertext Markup Language) Adalah bahasa yang di gunakan untuk membuat halaman website dan aplikasi web, HTML sendiri bukan bahasa pemrograman. Elemennya memberitahu browser cara menampilkan konten serta memberi label potongan konten seperti "ini judul tabel", "ini adalah tautan" dan lain-lain. Sedangkan frontednya merupakan bagian dari website yang terlihat di gunakan untuk mengatur tampilan dasar seperti struktur dan hiasan yang di lihat oleh user. Yang mana fronted ini terbagi beberapa jenis yaitu; HTML, CSS dan JavaScript. Bahwasanya dapat di artikan bahwa HTML adalah komponen utama yang memungkinkan kita untuk menentukan struktur situs web, dan CSS adalah sesuatu yang dilakukan untuk menentukan gaya dalam situs web, sedangkan JavaSript adalah bahasa pemrograman yang sering digunakan untuk mengontrol interaksi situs web baik kepada pengguna maupun server. Perbedaan front end dan back end developer bisa di lihat dari kemampuan yang diperlukan sampai waktu yang dibutuhkan untuk setiap perannya. Demikian rangkuman yang saya buat mengenai pembelajaran WEB.
